# Apple-style Notes — Demo

Простой статический проект заметок в стиле Apple (glassmorphism). Vanilla JS + Vite.

Как запустить:

1. Установить зависимости:

```bash
npm install
# или, если vite ещё не указан в package.json
npm install -D vite
```

2. Запустить dev-сервер:

```bash
npm run dev
```

3. Собрать production и предпросмотреть:

```bash
npm run build
npm run preview
# Открыть в браузере (macOS)
open http://localhost:4173
```

Файлы:
- `index.html` — разметка
- `styles.css` — стили
- `src/main.js` — модуль с логикой заметок

ОС: macOS (используйте команду `open` чтобы открыть URL).
